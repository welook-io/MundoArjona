# Mundo Arjona - Propuesta Comercial Kalia

Propuesta de implementación de Chat con IA para Mundo Arjona - Venta de paquetes VIP para shows de Ricardo Arjona.

## Contenido

Esta aplicación presenta una propuesta comercial para implementar un asistente de IA llamado "Mundo" que ayudará a:

- Aumentar la conversión en venta de paquetes VIP
- Brindar atención inmediata 24/7 a fans interesados
- Guiar personalizadamente hasta la compra del paquete ideal

## Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS
- **Typography**: Inter + Instrument Serif
- **Language**: TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) en el navegador.

## Producción

```bash
npm run build
npm start
```
